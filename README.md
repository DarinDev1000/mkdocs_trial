# MkDocs Trial

## Install

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install mkdocs mkdocs-material
```

## Activate environment on Windows Powershell

```powershell
.\.venv\Scripts\Activate.ps1
```

## Running on Windows
`mkdocs serve`

## Build Site
`mkdocs build`
